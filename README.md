## 📌 Featured Projects

### 📱 Project Connect  
**Android Student Project Collaboration App**  
🔗 [View Repository](https://github.com/HectorZouMingxuan/project-connect)

Project Connect is an Android mobile application that helps students discover projects, create teams, manage profiles, and communicate with teammates through a project-based chat system.

**Tech Used:** Kotlin, Jetpack Compose, Firebase Authentication, Cloud Firestore

**Key Features:**
- User registration and login using Firebase Authentication
- Profile creation and editing
- Project creation, joining, quitting, and deletion
- Real-time project list updates using Firestore
- Team chat with real-time messages
- Repository pattern and ViewModel-based state management
